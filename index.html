UI/UX

La interfaz de usuario o UI (User Interface) es un concepto que abarca arquitectura de información, patrones y diferentes elementos visuales que nos permiten interactuar de forma eficaz con sistemas operativos y softwares de diversos dispositivos. El diseño UX (User Experience) hace referencia a lo que experimenta el usuario antes, durante y después de entrar en contacto con una página web.
